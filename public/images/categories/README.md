# Изображения для категорий

## Требования к изображениям

- **Разрешение:** 1536 x 640 пикселей (широкоформатные баннеры)
- **Соотношение сторон:** 2.4:1
- **Формат:** JPG или PNG
- **Размер файла:** рекомендуется до 200 КБ для быстрой загрузки

## Список файлов

Разместите следующие изображения в этой папке:

1. `electronics.jpg` - Электроника (гаджеты, смартфоны, ноутбуки)
2. `furniture.jpg` - Мебель (кресла, столы, диваны)
3. `beauty.jpg` - Красота и здоровье (косметика, уход)
4. `sports.jpg` - Спорт и отдых (спортивный инвентарь)
5. `home.jpg` - Дом и сад (декор, инструменты)
6. `clothing.jpg` - Одежда (одежда, аксессуары)

## Рекомендации по дизайну

- Используйте качественные стоковые фото или рендеры
- Изображения должны иметь умеренную контрастность (текст будет белым поверх затемнения)
- Избегайте слишком ярких или перегруженных деталями картинок
- Основной объект должен быть в центре или слева от центра

## Где взять изображения

Бесплатные стоки:
- Unsplash: https://unsplash.com/
- Pexels: https://www.pexels.com/
- Pixabay: https://pixabay.com/

Поисковые запросы:
- "modern electronics gadgets"
- "minimalist furniture"
- "beauty cosmetics flat lay"
- "sports equipment"
- "home garden tools"
- "fashion clothing"

## Как выглядит в интерфейсе

Изображения используются как фон для кнопок категорий в выпадающем меню поиска.
Поверх изображения накладывается полупрозрачный черный overlay (40% непрозрачности),
чтобы белый текст был хорошо читаем.

При наведении overlay становится чуть темнее (50%) и появляется тень.
