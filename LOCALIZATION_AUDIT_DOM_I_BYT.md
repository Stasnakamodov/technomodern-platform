## SQL-ЗАПРОСЫ ДЛЯ МАССОВОГО ИСПРАВЛЕНИЯ

### Найти товары с китайскими иероглифами:
```sql
SELECT id, name FROM products
WHERE name ~ '[\u4e00-\u9fff]'
AND category_id IN (
  SELECT id FROM categories
  WHERE parent_id = '00000069-0000-0000-0000-000000690000'
);
```

### Найти товары с квадратными скобками:
```sql
SELECT id, name FROM products
WHERE name ~ '[\[\]【】]'
AND category_id IN (
  SELECT id FROM categories
  WHERE parent_id = '00000069-0000-0000-0000-000000690000'
);
```

### Найти товары с очень короткими названиями:
```sql
SELECT id, name FROM products
WHERE length(name) < 15
AND category_id IN (
  SELECT id FROM categories
  WHERE parent_id = '00000069-0000-0000-0000-000000690000'
```
